A high-performance, extensible chemical equation solver and reaction prediction platform, featuring a pluggable equation registry, and a modern PySide6 GUI.

> **Status**: Under active development.

---

## ✨ Features

- **🌍 Internationalization** — Built-in support for multiple languages (English, 中文, 日本語) with JSON-based locale files.
- **🧪 Extensible Architecture** — Clean separation of concerns: Hardware Abstraction Layer (HAL) → Kernel Registry → Engine Facade → Python Orchestration → GUI.

---

> **Note**: All equation implementations are currently **stubs** (return "Not implemented yet"). The registry framework is fully functional and ready for solver integration.
